# changelogithub-example

A project to practice using changelogithub

**Add Tag:**

```bash
git tag v*
```

**Push Tag:**

```bash
git push origin TAGNAME
# or
git push --tags
```

**Delete Tag:**

```bash
git tag -d TAGNAME
# or
git push --delete origin TAGNAME
```

**List Tag:**

```bash
git tag
```
